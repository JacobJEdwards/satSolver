parse :: Text -> Maybe Sudoku Source #

Parses a sudoku from a string. Returns Nothing if the input is invalid. Returns Just the sudoku otherwise.

>>> parse "1 2 3 -\n4 5 6 -\n7 8 9 -"
Just (Sudoku [[1, 2, 3], [4, 5, 6], [7, 8, 9]] 3)
